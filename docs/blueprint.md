# **App Name**: Nexo Financeiro

## Core Features:

- Visão Multicamadas: Dashboards alternáveis para finanças pessoais, visão compartilhada do casal e o módulo financeiro do escritório de advocacia.
- Automação de Pró-labore: Criação de transferências vinculadas entre despesa empresarial e receita pessoal com ID de transação unificado.
- Assistente de Categorização AI: Sugestão inteligente de categorias para novos lançamentos usando a descrição da transação como ferramenta de análise.
- Gestão de Movimentações Jurídicas: Lançamentos de escritório com suporte para nomes de clientes e referências de processos no módulo Business.
- Sincronização em Tempo Real via Firestore: Infraestrutura escalável no Firebase que garante dados atualizados instantaneamente para ambos os membros do casal.
- Arquitetura PWA Installable: Interface otimizada para dispositivos móveis com suporte a manifest e service worker para uso similar a um app nativo.
- Sistema de Convites e Membros: Módulo de onboarding para vincular cônjuges ao Household e configurar permissões automáticas por e-mail.

## Style Guidelines:

- Primary color: Sophisticated Deep Indigo (#8C96D9) conveying trust and professional integrity.
- Background: Charcoal Dark Blue (#22232B) for a premium dark-themed administrative aesthetic.
- Accent: Cool Cerulean (#ADCBEB) for highlighting active states, CTAs, and actionable items.
- Font pairing: 'Playfair' for sophisticated, authoritative headlines (serif); 'PT Sans' for legible, warm body text (sans-serif).
- Outline-style iconography with thin stroke weights, maintaining a clean and minimalist look suited for high-density financial data.
- Mobile-first bottom navigation bar featuring four main zones: Pessoal, Casal, Jurídico, and Ajustes.
- Smooth micro-interactions for financial charts and subtle transitions between account layers.
